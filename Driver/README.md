# MSP430_energy_toolbox
The personal MSP430 development toolbox based on Energia.

這是由 MSP430 控制器為核心，並透過了 Energia 軟體進行程式開發環境起源地專案。


##  Summary　（概要）
MSP430 是由德州儀器（Texas Instruments）開發的 16 位元 RISC 控制器，其主要特色為 低成本、低功耗，可廣泛應用於電池供電及其他嵌入式系統。
目前 MSP430 市面上有販售 G、F、FR系列，其各自差異特色如下表所示。


|系列| MSP430Gxxxx | MSP430Fxxxx | MSP430FRxxxx | 
|-----| ----- | ----- | ----- |
|記憶體| Flash + RAM | Flash + RAM | FRAM |
|功耗| 功耗低 | 功耗低 | 超低功耗 |
|特色| 成本低、適合學習 | 功能完整 | 寫入快、低功耗、耐久度高 |
